## Comienza en tres pasos

Empieza a crear facturas desde tickets con la API de Timbre.

### Paso 1: Obtén tu llave API

<AccordionGroup>
  <Accordion icon="key" title="Genera tu llave API">
    1. Inicia sesión en tu cuenta de Timbre en [api.usatimbre.com](https://api.usatimbre.com)
    2. Navega a la configuración de tu perfil
    3. Genera una nueva llave API o copia la existente

    <Warning>Mantén tu llave API segura y nunca la expongas en código del lado del cliente.</Warning>
  </Accordion>
  <Accordion icon="circle-check" title="Prueba tu llave API">
    Verifica que tu llave API funcione llamando al endpoint de perfil:

    ```bash
    curl -X GET "https://api.usatimbre.com/api/profile/TU_RFC" \
      -H "Authorization: Bearer tmb_tu_llave_api_aqui"
    ```

    <Tip>Reemplaza `TU_RFC` con tu RFC real y usa tu llave API real.</Tip>
  </Accordion>
</AccordionGroup>

### Paso 2: Configura tu perfil

<AccordionGroup>
  <Accordion icon="user" title="Verifica tu información fiscal">
    Antes de crear facturas, asegúrate de que tu información fiscal esté actualizada:

    ```bash
    curl -X GET "https://api.usatimbre.com/api/profile/TU_RFC/tax-info" \
      -H "Authorization: Bearer tmb_tu_llave_api_aqui"
    ```
  </Accordion>
  <Accordion icon="pen-to-square" title="Actualiza la información fiscal si es necesario">
    Actualiza tu información fiscal con los campos requeridos:

    ```bash
    curl -X PATCH "https://api.usatimbre.com/api/profile/TU_RFC/tax-info" \
      -H "Authorization: Bearer tmb_tu_llave_api_aqui" \
      -H "Content-Type: application/json" \
      -d '{
        "postal_code": "12345",
        "invoice_fiscal_regimen": "601",
        "invoice_cfdi_use": "G03"
      }'
    ```
  </Accordion>
</AccordionGroup>

### Paso 3: Crea tu primera factura

<Accordion icon="file-invoice" title="Crea factura desde ticket">
  Envía una imagen de ticket para crear una factura:

  ```bash
  curl -X POST "https://api.usatimbre.com/api/timbre/facturar-ticket" \
    -H "Authorization: Bearer tmb_tu_llave_api_aqui" \
    -H "Content-Type: application/json" \
    -d '{
      "receipt_image": "datos_imagen_base64",
      "cfdi_usage": "G03",
      "expense_description": "Gasto empresarial"
    }'
  ```

  <Note>Esta operación requiere 1 crédito. Asegúrate de tener créditos suficientes.</Note>
</Accordion>

## Códigos de Uso CFDI Comunes

| Código | Descripción |
|--------|-------------|
| `G01` | Adquisición de mercancías |
| `G02` | Devoluciones, descuentos o bonificaciones |
| `G03` | Gastos en general |
| `P01` | Por definir |

## Códigos de Régimen Fiscal Comunes

| Código | Descripción |
|--------|-------------|
| `601` | General de Ley Personas Morales |
| `603` | Personas Morales con Fines no Lucrativos |
| `605` | Sueldos y Salarios |
| `606` | Arrendamiento |
| `612` | Personas Físicas con Actividades Empresariales |
| `625` | Régimen Simplificado de Confianza |
