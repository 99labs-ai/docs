## Autenticación

Incluye tu llave API en cada solicitud:

```bash
Authorization: Bearer tmb_tu_llave_api
```

## Códigos HTTP

| Código | Significado |
|--------|-------------|
| `200` | Solicitud exitosa |
| `400` | Error en los datos enviados |
| `401` | Llave API inválida o faltante |
| `404` | Recurso no encontrado |
| `429` | Demasiadas solicitudes |
| `500` | Error interno del servidor |
